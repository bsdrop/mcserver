# mcserver

A personal Minecraft server fork based on [CanvasMC](https://github.com/CraftCanvasMC/Canvas), used for experimentation and a single hobby server.

Most changes in this fork are made with AI assistance, and much of the code has not been reviewed by a human. Use anything here at your own risk. This project is not affiliated with, endorsed by, or supported by the CanvasMC project, Paper, or Folia.

Please do not report issues from this fork to CanvasMC, Paper, or Folia. Issues here may be caused by local changes made in this fork.

Credit for the underlying server software goes to the CanvasMC team and its upstream projects, including Folia and Paper. See [LICENSE](LICENSE) for license details.
